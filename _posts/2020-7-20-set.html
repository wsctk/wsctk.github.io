Set集合方法

操作方法

在Set对象尾部添加一个元素。返回该Set对象：mySet.add(value);

从一个 Set 对象中删除指定的元素，成功删除返回 true，否则返回 false：mySet.delete(value);

返回一个布尔值来指示对应的值value是否存在Set对象中：mySet.has(value);

用来清空一个 Set 对象中的所有元素：mySet.clear();

遍历方法

返回一个元素类似[value,value]数组形式的迭代器对象，其中value是set集合对象中的每个元素：mySet.entries();

返回一个按照原set对象中元素插入顺序返回其所有元素的迭代器对象：mySet.values()/mySet.keys();

根据集合中元素的插入顺序，依次执行提供的回调函数：mySet.forEach(callback[, thisArg]);
